# 공통 용어 사전

> 도메인을 가리지 않고 워크스페이스 전체에서 쓰이는 용어입니다.
> 도메인별 용어는 `wiki/{도메인}/glossary.md`를 참조하세요.

| 용어 | 설명 |
|------|------|
| Command Center | 이 워크스페이스. 팀이 공용으로 사용하는 AI 작업 공간 |
| PRD | Product Requirements Document. 제품 요구사항 문서 |
| wiki/ | 도메인별 비즈니스 정책·용어·구현 상태를 정리하는 디렉토리 |
| ontology/ | 도메인 개념·관계·코드 위치를 YAML로 정의하는 디렉토리 |
| projects/ | clone한 실제 코드 레포지토리가 위치하는 디렉토리 |
